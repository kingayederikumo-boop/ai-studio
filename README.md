# AI Studio

A modern Next.js AI chat interface focused on NVIDIA models for high-performance inference.

## Features
- NVIDIA API integration with model selection
- Streaming chat responses
- GitHub repo management
- Supabase chat history
- Responsive UI with Tailwind

## Quick Start
1. `npm install`
2. Copy `.env.local.example` to `.env.local` and configure keys
3. `npm run dev`

## Models
See NVIDIA models in env and nvidiaService.ts

Deploy to Vercel.
